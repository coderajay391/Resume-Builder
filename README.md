A simple resume builder project to help you create structured, professional resumes quickly.

## Features

- Form-based resume creation
- Preview/preview-friendly layout
- Export/print-ready output (depending on the implementation)

## Getting Started

### Install

```bash
npm install
```

### Run

```bash
npm start
```

or

```bash
npm run dev
```

## How to Use

1. Fill in your details (name, contact, summary, experience, education, skills).
2. Use the live preview to verify formatting.
3. Export/print the final resume (if supported).
